About this Project :
--------------------

This is a QR code generater project which you can generate 6 different types of qr code and also you can save the generated qr code in your local computer

Types of Qr code you can generate :
-----------------------------------

1. Text QR code
2. URL QR code
3. Save mobile no Contact QR code
4. send sms QR code
5. connect a wifi network QR code
6. make a phone call QR code


Dependencies :
--------------

Python tkinter library
